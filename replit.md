# Sunlit

Static site server (Node.js/Express-style with raw http) that serves index.html, games, movies, and apps. Includes a movie URL proxy.

## Run
- Workflow: `Start application` runs `PORT=5000 node server.js`
- Server binds to 0.0.0.0:5000

## Deployment
- Autoscale, run: `node server.js` with PORT=5000
